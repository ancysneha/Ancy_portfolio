## üéØ Customization

Update the following to personalize:

- **Personal Info**: Edit `src/components/Hero.jsx` for name and role
- **About Section**: Modify `src/components/About.jsx`
- **Skills**: Update skills array in `src/components/Skills.jsx`
- **Projects**: Edit projects array in `src/components/Projects.jsx`
- **Experience**: Update experience data in `src/components/Experience.jsx`
- **Contact**: Modify contact links in `src/components/Contact.jsx`
- **Colors**: Adjust color scheme in `tailwind.config.js`
